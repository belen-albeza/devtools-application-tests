Basic Service Worker

Empty service worker, only gets registered.

  1. Open the Application panel and go to the Service Workers page.
  2. A You need to register a Service Worker message will appear, with some help text.
  3. Click on this button
  4. A Service Worker registration should appear, with a matching scope (ending in service-workers/basic) and the sw.js source URL should appear
  5. When the worker is under the status Running, click on the Debug button
  6. The debugger should open. Go back to the Application panel
  7. Click on the Unregister button in the Application panel to clean up
